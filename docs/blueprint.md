# **App Name**: SprintSync Board

## Core Features:

- Kanban Board View: Display tasks organized into customizable columns (e.g., New, In Development, Resolved, Closed) for clear project tracking.
- Task Creation & Editing: Allow users to create new tasks, add titles, detailed descriptions, assignees, due dates, and update their status.
- Drag-and-Drop Task Workflow: Enable intuitive drag-and-drop functionality to move tasks between different columns on the board, reflecting status changes.
- User Assignment: Assign tasks to specific team members for clear ownership and accountability.
- AI Task Detail Generator: A generative AI tool that can assist users in elaborating on brief task titles into full descriptions, suggesting sub-tasks, or generating acceptance criteria.
- Quick Task Filtering: Filter tasks by assignee, status, or search for keywords to quickly locate relevant items within a board.

## Style Guidelines:

- Primary color: A professional and clean dark blue (#174799) to signify trust and stability in a technical context.
- Background color: A subtle, desaturated light blue-gray (#F2F5FA) providing a calming and focused workspace, harmonizing with the primary blue.
- Accent color: A vibrant purple (#5526E5) for highlights, interactive elements, and indicating active states, offering a modern contrast.
- Headline and body font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for data-dense interfaces.
- Utilize a consistent set of minimal, clean outline icons for navigation and actions, ensuring clarity without visual clutter.
- Employ a responsive grid-based layout for the Kanban board columns, allowing for clear separation of task states and efficient organization.
- Incorporate subtle and swift animations for card movements and status changes, enhancing user feedback and interface fluidity without being distracting.
